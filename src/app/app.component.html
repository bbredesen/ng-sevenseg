Seven Segment Display Demo

Basic usage: <seven-seg value="6">
Multiple digits: <seven-seg value="3" digits="3">, should display "_ _ 3"
No value: <seven-seg digits="3">, should render "off"
Empy value: <seven-seg digits="3" value="">, interpreted as zero
Zero value: <seven-seg digits="3" value="0">
One decimal place: <seven-seg value="3.6" digits="3" decimalPlaces="1">, should display "_ 3.6"
Two decimal places with rounding: <seven-seg value="3.658" digits="3" decimalPlaces="2">, should display "3.66"
Leading null, internal zero, trailing zero: <seven-seg value="304.6" digits="6" decimalPlaces="2">, should display "_ 304.60"
